📖 Theory

A Multilayer Perceptron (MLP) is a feedforward neural network with one or more hidden layers. Unlike a single perceptron, MLPs can learn non-linear decision boundaries. The architecture consists of an input layer (784 neurons for 28×28 images), two hidden layers (256 and 128 neurons with ReLU activation and Dropout), and an output layer (10 neurons with Softmax). Training uses Adam optimizer with Cross-Entropy loss.
