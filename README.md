# Deep Learning Experiments

This repository contains a collection of 12 self-contained deep learning experiments, exploring various architectures and applications of neural networks. Each experiment includes a complete project setup with a Flask backend and a modern glassmorphism frontend for easy interaction.

## 🧪 Experiments

1. **Experiment 01: ANN Basics** - Introduction to Artificial Neural Networks.
2. **Experiment 02: MLP Classification** - Multi-Layer Perceptron for classification tasks.
3. **Experiment 03: CNN Image Classification** - Convolutional Neural Networks for classifying images.
4. **Experiment 04: CNN Facial Emotion** - Detecting facial emotions using CNNs.
5. **Experiment 05: Transfer Learning Emotion** - Leveraging pre-trained models for emotion recognition.
6. **Experiment 06: RNN Sentiment Analysis** - Recurrent Neural Networks for analyzing text sentiment.
7. **Experiment 07: Image Captioning** - Generating textual descriptions for images using CNN + RNN architectures.
8. **Experiment 08: POS Tagging** - Part-of-Speech tagging for Natural Language Processing.
9. **Experiment 09: LSTM Text Generation** - Generating text sequences using Long Short-Term Memory networks.
10. **Experiment 10: GRU Sentiment** - Sentiment analysis using Gated Recurrent Units.
11. **Experiment 11: Attention Case Study** - Deep dive into Attention mechanisms.
12. **Experiment 12: Transformers Case Study** - Exploring modern Transformer models for advanced NLP tasks.

## 🚀 Getting Started

You can navigate into any experiment's folder to run it individually. Alternatively, you can use the `start_all.sh` script provided in the root directory to manage and launch the services.
