📖 Theory & Architecture

Network Architecture: 2 → 4 → 1

This ANN solves the XOR problem which is not linearly separable, requiring at least one hidden layer. The network uses:

  • Input Layer: 2 neurons (X1, X2)
  • Hidden Layer: 4 neurons with Sigmoid activation
  • Output Layer: 1 neuron with Sigmoid activation
  • Learning: Backpropagation with gradient descent

Backpropagation Algorithm

1. Forward pass computes activations through each layer.
2. Error is calculated at the output (MSE).
3. Gradients are propagated backward using the chain rule.
4. Weights and biases are updated using gradient descent.
