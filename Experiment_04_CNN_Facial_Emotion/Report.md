# CNN — Facial Emotion Detection\n\n## Aim\nCustom Convolutional Neural Network for classifying human facial emotions from grayscale images.\n\n## Theory\nFacial emotion recognition uses a deep CNN architecture (Conv2D -> ReLU -> MaxPool2D) to extract spatial features from 48x48 pixel images. The network classifies the image into one of 7 emotion categories: Angry, Disgust, Fear, Happy, Sad, Surprise, Neutral.
